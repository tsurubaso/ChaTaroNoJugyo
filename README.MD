## この授業のゴール

この授業の最終目標は：

**ユーザーのタスクに従って動けるAIエージェントを作ることです。**

つまり：

- ユーザーが「これやって」と言う  
- AIが理解する  
- AIが行動する  
- タスクを完了する  

そういう仕組みを作れるようになることです。

---

## 大事なルール

この授業では、次のことを大切にします：

### ✅ 分からないことは「分からない」と言う
